# GAIT試験 学習ガイド

> GAIT（Global Assessment of Information Technology）試験対策のための学習資料です。

## このサイトについて

GAIT試験の学習に必要な情報をまとめています。

- **試験概要** - GAITとは何か、試験の種類と特徴
- **出題範囲** - 8つの技術分野の詳細
- **カテゴリ別学習** - 各分野の重要ポイントを解説
- **チートシート** - 試験直前の確認用
- **クイズアプリ** - 80問の4択問題で実践的に学習

## クイックスタート

### 1. 試験について知る

まず [GAIT試験とは](GAIT試験調査結果.md) を読んで、試験の全体像を把握しましょう。

### 2. 出題範囲を確認

GAIT2.0の出題範囲は以下の8分野です：

| 分野 | 主な内容 | 学習ページ |
|------|---------|-----------|
| インフラストラクチャ | RAID、ネットワーク、サーバー | [学習する](infrastructure.md) |
| OS・ミドルウェア | Windows、Linux、パーミッション | [学習する](os-middleware.md) |
| データベース | SQL、正規化、ACID | [学習する](database.md) |
| アプリケーション | OOP、REST、Git、テスト | [学習する](application.md) |
| クラウド | IaaS/PaaS/SaaS、コンテナ | [学習する](cloud.md) |
| セキュリティ | 暗号化、認証、攻撃手法 | [学習する](security.md) |
| DX技術 | AI、IoT、ブロックチェーン | [学習する](dx-technology.md) |
| DX利活用 | DX戦略、KPI、プロジェクト管理 | [学習する](dx-utilization.md) |

### 3. クイズで練習

[クイズアプリ](http://localhost:3000/) で実際の問題形式に慣れましょう。

- **80問収録** - 8カテゴリ各10問
- **模擬試験モード** - GAIT2.0/e-GAIT2.0形式で本番対策
- **お気に入り機能** - 重要な問題をブックマーク
- **復習モード** - 間違えた問題を自動で復習リストに追加
- **学習ダッシュボード** - 進捗・苦手分野を可視化

### 4. 試験直前の確認

[チートシート](cheatsheet.md) で重要ポイントを最終確認しましょう。
