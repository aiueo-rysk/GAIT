# GAIT試験 学習ガイド

> GAIT（Global Assessment of Information Technology）試験対策のための学習資料です。

## このサイトについて

GAIT試験の学習に必要な情報をまとめています。

- **試験概要** - GAITとは何か、試験の種類と特徴
- **出題範囲** - 7つの技術分野と20カテゴリの詳細
- **対策方法** - 効率的な学習方法と推奨教材
- **クイズアプリ** - 4択問題で実践的に学習

## クイックスタート

### 1. 試験について知る

まず [GAIT試験とは](GAIT試験調査結果.md) を読んで、試験の全体像を把握しましょう。

### 2. 出題範囲を確認

GAIT2.0の出題範囲は以下の7分野です：

| 分野 | 主な内容 |
|------|---------|
| インフラストラクチャ | ストレージ、ネットワーク、サーバー |
| OS・ミドルウェア | Windows、Linux、UNIX |
| アプリケーション | 設計、実装、開発プロセス |
| クラウド | SDx、マネージドサービス |
| セキュリティ | 要素技術、インフラ・クラウド |
| DX技術 | AI、IoT、fog |
| DX利活用 | 技術利活用、データ利活用 |

### 3. クイズで練習

[クイズアプリ](/quiz-app/) で実際の問題形式に慣れましょう。

- カテゴリ別に学習可能
- 間違えた問題を自動で復習リストに追加
- 苦手分野を可視化
