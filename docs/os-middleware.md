# OS・ミドルウェア

オペレーティングシステムとミドルウェアに関する知識です。

## 出題範囲

- Windows
- Linux
- UNIX

---

## オペレーティングシステム（OS）

### OSの基本機能

| 機能 | 説明 |
|------|------|
| プロセス管理 | プログラムの実行制御 |
| メモリ管理 | 物理/仮想メモリの割当 |
| ファイル管理 | ファイルシステムの操作 |
| デバイス管理 | 入出力デバイスの制御 |
| ユーザー管理 | 認証・権限管理 |

### カーネル

OSの中核部分。ハードウェアとアプリケーションの橋渡しをする。

| 種類 | 説明 | 例 |
|------|------|-----|
| モノリシックカーネル | 全機能を1つのカーネルに | Linux |
| マイクロカーネル | 最小限の機能のみ | Minix |
| ハイブリッドカーネル | 両者の中間 | Windows NT |

## Windows

### Active Directory（AD）

Windowsネットワークの中核となるディレクトリサービス。

| 概念 | 説明 |
|------|------|
| ドメイン | 管理単位（ユーザー・コンピュータのグループ） |
| ドメインコントローラ | ADを管理するサーバー |
| フォレスト | 複数ドメインの最上位集合 |
| OU | Organizational Unit（組織単位） |
| グループポリシー | 設定を一括適用 |

### Windowsの主要サービス

| サービス | 説明 |
|---------|------|
| IIS | Internet Information Services（Webサーバー） |
| WSUS | Windows Server Update Services（更新管理） |
| Hyper-V | 仮想化プラットフォーム |
| DNS | ドメイン名解決サービス |
| DHCP | IPアドレス自動割当サービス |

### レジストリ

Windowsの設定情報を格納するデータベース。

| ハイブ | 内容 |
|--------|------|
| HKEY_LOCAL_MACHINE | システム全体の設定 |
| HKEY_CURRENT_USER | 現在のユーザー設定 |
| HKEY_CLASSES_ROOT | ファイル関連付け |
| HKEY_USERS | 全ユーザーの設定 |

### ファイルシステム

| ファイルシステム | 特徴 |
|-----------------|------|
| NTFS | 暗号化、圧縮、アクセス権対応 |
| FAT32 | 互換性が高い、4GB制限 |
| exFAT | FAT32の拡張、大容量対応 |
| ReFS | 耐障害性向上（新世代） |

### パーミッション

```
rwxrwxrwx
│││││││││
│││││││└─ 他ユーザー実行
│││││││── 他ユーザー書込
││││││─── 他ユーザー読取
│││││──── グループ実行
││││───── グループ書込
│││────── グループ読取
││─────── 所有者実行
│──────── 所有者書込
───────── 所有者読取
```

| 数値 | 権限 |
|------|------|
| 7 | rwx（読取+書込+実行） |
| 6 | rw-（読取+書込） |
| 5 | r-x（読取+実行） |
| 4 | r--（読取のみ） |
