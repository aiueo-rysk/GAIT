# GAIT試験 調査結果

## GAITとは

GAIT（Global Assessment of Information Technology）は、JTP株式会社が開発したITスキルを数値化・可視化するアセスメントです。

- **合否判定ではなく、現在のITスキルレベルを客観的に測定する試験**
- 2012年にリリース、2021年4月にGAIT2.0へバージョンアップ
- 累計15万人以上、600社以上で活用
- 特定の国・企業・団体に依存しない国際的なIT試験
- 問題は最新の技術動向に合わせて毎年更新

---

## 試験の種類

| 項目 | GAIT2.0 | e-GAIT2.0 |
|------|---------|-----------|
| 対象 | ITエンジニア向け | 全ビジネスパーソン向け |
| 問題数 | 160問 | 80問 |
| 試験時間 | 60分 | 30分 |
| 満点 | 990点 | 300点 |
| 受験料 | 9,350円（税込） | 3,300円（税込） |
| 難易度 | 高い（「使いこなせる」レベル） | 低い（前提知識を問う） |
| 形式 | 4択選択式 | 4択選択式 |

## スコア評価（GAIT2.0）

| スコア | 評価 | 意味 |
|--------|------|------|
| 700点以上 | ゴールド | 十分なITスキルを保有 |
| 480〜699点 | シルバー | 基礎理解あり、弱点克服が望ましい |
| 479点以下 | ブロンズ | スキルの底上げが必要 |

---

## 試験の特徴

- IT用語の知識を問う問題が多い
- 4択形式が基本、マルチチョイス問題も出題
- VMware、Oracle、IIS等ベンダー製品に関する問題が含まれる
- オンライン自宅受験（監督なし）
- 1問あたり約22秒（GAIT2.0の場合）で解答が必要

## 対策方法

### 推奨教材

GAIT専用教材は廃版のため、以下が有効：

- **基本情報技術者試験の参考書**
- **応用情報技術者試験の参考書**
- IT用語辞典・用語集

### 重点対策分野

| 分野 | 具体的な内容 |
|------|-------------|
| OS関連 | Windows、Active Directory、Linux基礎 |
| データベース | Oracle、SQL、ER図、用語（選択・射影等） |
| ネットワーク | TCP/IP、プロトコル、ルーティング |
| クラウド・仮想化 | VMware、AWS/Azure/GCPの基礎概念 |
| セキュリティ | 暗号化、認証、ファイアウォール |

### 学習ステップ

```
Step 1: 現状把握
  └ どの試験（GAIT2.0 or e-GAIT2.0）を受けるか確認
  └ 受験日を決める

Step 2: 基礎固め
  └ 基本情報技術者の参考書で7分野を網羅的に学習
  └ 各分野の基本用語を理解

Step 3: 弱点補強
  └ 苦手分野を集中的に学習
  └ ベンダー製品（Oracle、VMware等）の基礎知識

Step 4: 用語・略語の暗記
  └ IT用語が多く出題されるため、用語集で補強
  └ 略語の正式名称と意味を覚える

Step 5: 時間配分の練習
  └ 160問/60分 = 1問あたり約22秒
  └ わからない問題は飛ばす判断力を養う
```

### 受験時の注意点

- 時間配分が非常に重要（1問22秒ペース）
- わからない問題は飛ばして後で戻る
- ベンダー固有の知識は深追いしすぎない
- 得意分野で確実に得点する
