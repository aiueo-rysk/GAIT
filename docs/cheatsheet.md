## インフラストラクチャ

### RAID早見表
| RAID | 特徴 | 最低台数 | 冗長性 |
|------|------|---------|--------|
| 0 | ストライピング | 2 | なし |
| 1 | ミラーリング | 2 | 1台OK |
| 5 | パリティ分散 | 3 | 1台OK |
| 6 | 二重パリティ | 4 | 2台OK |
| 10 | ミラー+ストライプ | 4 | 各組1台OK |

### ストレージ種類
- **DAS**: 直接接続
- **NAS**: ファイル共有（ネットワーク）

## データベース

### SQL基本
```sql
SELECT / INSERT / UPDATE / DELETE  -- DML
CREATE / ALTER / DROP             -- DDL
GRANT / REVOKE                    -- DCL
COMMIT / ROLLBACK                 -- トランザクション
```

### 正規化
- **第1正規形**: 繰り返し項目なし
- **第2正規形**: 部分関数従属なし
- **第3正規形**: 推移関数従属なし

### ACID特性
- **A**tomicity（原子性）
- **C**onsistency（一貫性）
- **I**solation（独立性）
- **D**urability（永続性）

### JOIN種類
- `INNER JOIN`: 両方に存在
- `LEFT JOIN`: 左テーブル基準
- `RIGHT JOIN`: 右テーブル基準
